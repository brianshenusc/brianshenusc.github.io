Stock Algorithms

Automated Mean Reversion Algorithm with Random Forest Classifier

About

This project involves the development of an automated stock trading strategy using a Random Forest Classifier integrated with a mean reversion approach. The strategy is designed to detect trading opportunities where stock prices are expected to revert to their average, leveraging machine learning to identify patterns and improve decision-making for profitable trades through extensive feature selection.

Analysis (Note: The algorithm is constantly running, so these metrics are subject to change)

  • Beta: The lower beta indicates that the strategy has less exposure to market volatility, suggesting reduced risk compared to the S&P500.
  • Sharpe Ratio: The Sharpe ratio reflects a favorable risk-adjusted return, indicating that the strategy is efficiently balancing risk and return
  • Win Rate: The high win rate demonstrates the model's effectiveness in making profitable trades.

Technology Used

  • Python: This project was coded in a Jupyter Notebook using numpy, pandas, yfinance, scikit-learn, plotly, and alpaca_trade_api.
  • AWS: The algorithm is deployed in an AWS ECR repository via Docker. It is run hourly during market hours through AWS Lambda, and trades are recorded in AWS DynamoDB. The interactive Plotly graph above is generated using these DynamoDB records and hosted on an AWS S3 bucket, which is also automatically updated hourly
  • Machine Learning: The algorithm utilizes a Random Forest Classifier Model with careful feature selection, which increased the win rate from 72% to 81% when back tested.
  • Alpaca Markets: This project leverages Alpaca Market's free web API to conduct latency-free trades
